Ink Elements Documentation

Ink Elements is a Polymer library of custom elements for publishing documents via HTML and CSS. Features include:

  • Page References

Note: Ink Elements require the paged.js Paged Media polyfill.

Getting Started

Install the component:

$ npm install ink-elements --save

Then import polyfills:

<script src="node_modules/@webcomponents/webcomponentsjs/webcomponents-loader.js"></script>
<script src="https://unpkg.com/pagedjs/dist/paged.polyfill.js"></script>

Import custom element:

<script type="module" src="node_modules/ink-elements/ink-doc.js">

Page References

You can reference another page with the ink-page-ref component:

<p>See page <ink-page-ref ref="html-id"></ink-page-ref> for more information. </p>

See page for more information.

Publishing

To publish as a PDF the simplest, most reliable way is via Chrome's PDF preview feature:

  1. Open in Chrome
  2. Click File -> Print...
  3. Click on `More settings`
  4. Set `Paper size` to `A4`
  5. Set `Margins` to `None
  6. Click in `Open PDF in Preview`
  7. Then click File -> Save...
