Ink Elements Documentation

Ink Elements is a Polymer library of custom elements for publishing documents via HTML and CSS. Features include:

  • Printable
  • Page Numbering
  • Page References

Getting Started

Install the component using Bower:

$ bower install ink-elements --save

Then import the polyfill:

<script src="bower_components/webcomponentsjs/webcomponents-lite.min.js"></script>

Import custom element:

<link rel="import" href="bower_components/rgladwell/ink-doc.html">

Usage

An Ink document starts with a ink-doc root element:

<ink-doc></ink-doc>

Each document contains one-or-more pages:

<ink-doc>
  <ink-page>
  </ink-page>

  <ink-page>
  </ink-page>
</ink-doc>

Publishing

To publish as a PDF the simplest, most reliable way is via Chrome's PDF preview feature:

  1. Open in Chrome
  2. Click File -> Print...
  3. Click on `More settings`
  4. Set `Paper size` to `A4`
  5. Set `Margins` to `None
  6. Click in `Open PDF in Preview`
  7. Then click File -> Save...
